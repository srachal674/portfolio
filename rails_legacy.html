Temperature Conversion

A script that takes a temperature in Celsius, converts it to Fahrenheit and prints the result, illustrating user input and arithmetic operations.


    def c_to_f 
      puts "Enter degrees Celsius" 
      degrees = gets 
      fh = (degrees.to_i * 9/5) + 32 
      puts "#{degrees} Celsius is #{fh} Fahrenheit."
    end
    
    puts c_to_f

Ordinal Challenge

Converts plain numbers into their ordinal form (1 → 1st, 2 → 2nd, etc.), demonstrating conditional logic. 


    puts "Enter a whole number"
    number = gets.chomp
    answer = number.to_i
    age = number % 10
                  
    def answer(age)
      if age == 1
        return "#{age} st"
      elsif age == 2
        return "#{age} nd"
      elsif age == 3
        return "#{age} rd"
      else
        return "#{age} th"
      end
    end
    
    puts answer(age)

Foobar

An adaptation of the classic FizzBuzz problem: given a number of items, it prints “Foo,” “Bar” or “Foobar” depending on divisibility, highlighting looping and conditionals.


    puts "How many items do you want to see?"
    items = gets.chomp.to_i
                  
    list = []
    (1..items).each do |x|
      if x % 3 == 0 && x % 5 == 0
        list << "Foobar"
      elsif x % 5 == 0 
        list << "Bar"
      elsif x % 3 == 0
        list << "Foo"
      else 
        list << x
      end
    end
    puts list
